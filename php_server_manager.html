üìã Manual PHP Server Setup Instructions:

If the automatic checks fail, follow these steps:

Option 1: Using Command Prompt

1. Open Command Prompt as Administrator
2. Navigate to project: cd "d:\Documents\GitHub\Agrilink-Agri-Marketplace"
3. Start PHP server: php -S localhost:8080
4. Keep the command prompt open while using the app

Option 2: Using XAMPP/WAMP

1. Install XAMPP or WAMP
2. Copy your project to htdocs folder
3. Start Apache service
4. Use URL: http://localhost/Agrilink-Agri-Marketplace/backend/

Option 3: Using the Batch File

1. Double-click "start_php_server.bat" in your project root
2. A command window will open with the PHP server
3. Keep it running while using the application
